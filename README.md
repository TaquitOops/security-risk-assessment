# Security Risk Assessment Report 

This repository contains a **security risk assessment report** and practical examples of **network hardening methods** applied to a scenario involving a data breach in a social media organization.

##  Repository Structure

- `docs/` → Original course documents:
  - **Security-risk-assessment-report.docx**
  - **Network-hardening-tools.xlsx**
- `src/examples/` → Written examples of vulnerabilities and solutions in English and Spanish.
- `README.md` → This file with the main description.
- `LICENSE` → Project license.
- `CONTRIBUTING.md` → Guidelines for contributing.

##  Hardening Tools and Methods Included

- Password policies  
- Multifactor authentication (MFA)  
- Firewall maintenance and port filtering  
- Secure database configuration  
- Removal of unused applications and services  

##  Bilingual Versions

The repository includes examples in **English** and **Spanish** to support international learning environments.
