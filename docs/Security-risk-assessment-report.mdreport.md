# Security Risk Assessment Report 

## Part 1: Select up to three hardening tools and methods to implement

1. Password Policies  
2. Multifactor Authentication (MFA)  
3. Firewall Maintenance and Port Filtering  

---

## Part 2: Explain your recommendations

### Password Policies
- **Effectiveness:** Prevents weak or shared passwords, reducing unauthorized access risks.  
- **Frequency:** Enforced continuously, reviewed quarterly.  

### Multifactor Authentication (MFA)
- **Effectiveness:** Adds an extra layer of security, protecting against credential theft and brute-force attacks.  
- **Frequency:** Implemented once, maintained with periodic reviews.  

### Firewall Maintenance and Port Filtering
- **Effectiveness:** Controls network traffic and blocks suspicious or malicious access.  
- **Frequency:** Rules should be reviewed monthly or after any security incident.
